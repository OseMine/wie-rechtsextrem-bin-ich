# Wie rechtsextrem bin ich?

Interaktiver Fragebogen zu rechtsextremen Einstellungen auf Basis der **Rechtsextremismus-Skala** der Bertelsmann Stiftung (EINWURF – Zukunft der Demokratie 01.2021).

## Test

18 Aussagen in 6 Dimensionen – eine Frage nach der anderen mit 5-stufiger Likert-Skala.

### Dimensionen

1. Befürwortung einer rechtsgerichteten Diktatur
2. Chauvinismus
3. Verharmlosung des Nationalsozialismus
4. Fremdenfeindlichkeit
5. Antisemitismus
6. Sozialdarwinismus

### Auswertung

- **Pro Dimension** (3 Items, 3–15 Punkte): < 9 nicht rechtsextrem, 9–11 latent, ≥ 12 manifest
- **Gesamt** (18 Items, 18–90 Punkte): < 54 nicht rechtsextrem, 54–62 latent, ≥ 63 manifest

## Nutzung

Einfach `index.html` im Browser öffnen. Kein Server nötig.

Online-Version: [wie-rechtsextrem-bin-ich-iota.vercel.app](https://wie-rechtsextrem-bin-ich-iota.vercel.app/)

## Quelle

Bertelsmann Stiftung (2021). *EINWURF – Zukunft der Demokratie 01.2021: Rechtsextreme Einstellungen der Wähler:innen vor der Bundestagswahl 2021*.
